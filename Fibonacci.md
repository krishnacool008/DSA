# Recursion

### 1. State
`f(i)`
---

---

### 2. English Meaning
`f(i) represents number at ith position in a fibonacci series`
---

---

### 3. Choices / Contribution
- Either we can have decision problem where choice matters
- Or we can have contribution of different states to get the result
    #### Contribution 1: `f(i-1)`
    #### Contribution 2: `f(i-2)`
---

### 4. Recurrence
`f(i) = f(i-1) + f(i-2)`
---
Both contribution is needed, this is contribution problem

---

### 5. Verification
- f(i) represent number at ith position in fibonacci series
- f(i-1) represent number at i-1th position in fibonacci series
- f(i-2) represent number at i-2end position in fibonacci series

---


### 6. Base Case
`f(0) = 0`
`f(1) = 1`
